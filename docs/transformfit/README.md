## The 30-Day Arc

### Week 1 (Days 0-7): Trust Formation
Coach builds a real relationship. User completes 3+ workouts. Ends with the Week 1 Trophy and a natural subscription ask.

**Retention target:** 40% Day 7 retention (vs 15-20% industry average)

### Week 2 (Days 8-14): Habit Seeding
Coaching conversation becomes part of daily rhythm. LAAL protocol activates. Variable rewards at Days 11 and 14.

**Retention target:** 30% Day 14 retention

### Week 3 (Days 15-21): Friction Test
User navigates first real obstacle with the coach. Recovery loop design. Day 21 "3-week consistent" milestone.

**Retention target:** 20% Day 21 retention

### Week 4 (Days 22-30): Identity Consolidation
"I train with my coach" becomes identity. Community activation. Day 30 reflection closes the arc.

**Retention target:** 35% Day 30 retention (vs 8-12% industry average)

## The Adaptive Coaching Escalation

The coach gets progressively more direct as trust builds:

- **Days 0-2:** Warm, curious, unhurried
- **Days 3-6:** Comfortable, occasional teasing, can push harder
- **Days 7-13:** Direct, playful, can challenge choices
- **Days 14-21:** Honest, frank, difficult conversations possible
- **Days 22-30:** Peer energy, co-creating the program

## Key Behavioral Mechanics Reference

| Mechanism | First Used | How |
|-----------|-----------|-----|
| Implementation Intention | Day 0 | "When [time], I will [action]" locks first workout |
| Commitment Device | Day 0 | User states their "why" in their own words |
| Endowment Effect | Day 1+ | They've invested in answers — leaving = losing |
| Identity Framing | Day 1 | "You're someone who shows up" |
| Variable Reward | Days 4, 7, 11, 14, 21, 28 | Unexpected personal insights |
| Streak Psychology | Day 4 | Mentioned conversationally, never gamified |
| LAAL Protocol | Day 7+ | Loss framing only after gains established |
| Social Proof | Day 3, 7 | Natural references to "people who..." |
| Reciprocity | Day 5 | Coach gives first; asks deeper question later |
| Future Self Anchor | Day 4, 7, 21, 30 | Concrete near-term future |

## Script Format Reference

Each day script follows this structure:

1. **Overview** — Where user is, what coach is accomplishing, trust level
2. **The Script** — Exact iMessage bubbles with all branching paths
3. **Behavioral Mechanics at Work** — Each mechanism explained
4. **Moments of Truth** — (Day 0 has all 7; others have relevant ones)
5. **Retention Risk** — Specific churn risk + how script mitigates
6. **Setup for Next Day** — What hook is planted
7. **Metrics Target** — Retention target + optimization goal
