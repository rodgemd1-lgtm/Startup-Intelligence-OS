# Pull Request Template

## Objective

What business / operating problem does this PR solve?

## Scope

What is intentionally included?
What is intentionally excluded?

## Files changed

List the main file groups changed.

## Why this is the right slice

Why is this a reviewable increment instead of a larger rewrite?

## Validation

What was run or checked?
- tests
- lint
- schema validation
- launcher sanity check
- local app run

## Risks

What could break?
What still depends on follow-up work?

## Follow-up

What should the next PR do?
