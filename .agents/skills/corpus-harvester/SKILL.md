name: corpus-harvester
description: Use when building or expanding a large local corpus from binary documents, repos, crawls, locators, or generated extraction outputs.
---

Produce or update:
- source inventory
- extraction plan
- manifest summary
- locator file
- ingestion path
- failure log

Prefer:
- local-first harvesting before live crawl
- provenance and namespace separation
- resumable manifests and extraction scripts
