## Phases

### Phase 1: Weekly Strategic Digest Agent (15 min)

**What:** Agent that produces a Friday synthesis: what happened this week, what's next, decisions pending, signals detected. Composes output from ARIA briefs, SCOUT signals, HERALD responses, PATTERN-MATCHER findings, and LEDGER metrics.

**Why:** The V3 spec calls for "Automated Friday synthesis: what happened, what's next, decisions pending." We have all the source agents — this one just synthesizes them.

**Agent:** `.claude/agents/digest.md`

**Output:** `.startup-os/briefs/digest-{date}.md`

**Inputs (reads from):**
- `.startup-os/briefs/aria-*.md` (last 7 days)
- `.startup-os/briefs/scout-*.md` (last 7 days)
- `.startup-os/briefs/herald-*.md` (last 7 days)
- `.startup-os/briefs/patterns-*.md` (last 7 days)
- `.startup-os/briefs/ledger-*.md` (last 7 days)

**Validation:** Run the agent against whatever briefs exist. Produces structured output with all 5 sections.

**Autonomy:** T0 (new agent, needs observation period)

### Phase 2: Governance Agents — Anti-fragility + Strategic Optionality (20 min)

**What:** Two lightweight governance agents that complete V3's Layer 6.

#### 2a: Anti-fragility Monitor

Scans for signals that the system is making things WORSE:
- Agent outputs that required heavy human editing (low quality)
- Scheduled tasks that produce stale/irrelevant output
- Workflows that add noise without signal
- Automation that creates more work than it saves

**Agent:** `.claude/agents/antifragility-monitor.md`
**Output:** `.startup-os/briefs/antifragility-{date}.md`
**Schedule:** Weekly (run alongside PATTERN-MATCHER)

#### 2b: Strategic Optionality Preservation

Scans recent decisions and flags any that close doors:
- Technology choices that create lock-in
- Pricing/licensing commitments
- Architecture decisions that are hard to reverse
- Partnership/vendor dependencies

**Agent:** `.claude/agents/optionality-scout.md`
**Output:** `.startup-os/briefs/optionality-{date}.md`
**Schedule:** On-demand (triggered when major decisions are made)

**Validation:** Both agents produce structured output. Anti-fragility identifies at least one area to watch (or reports "system healthy"). Optionality scout identifies reversibility of recent decisions.
