# Phase 1: Research Questions — AI Video Generation for Social Media Film Studio

**Date:** 2026-03-23
**Project:** Social Media Film Studio (Viral Architect capability)
**Use Cases:** TransformFit brand content, Mike's personal Instagram, cinematic short-form
**Requirement:** Ultra-realistic, professional-grade, NOT cartoon/fake-looking

---

## Category 1: Tool Landscape & Realism Quality

1. What are the top AI video generation tools as of March 2026, ranked by photorealism?
2. Which tools produce output that passes the "would someone scroll past this thinking it's real footage" test?
3. How do these tools compare on human motion realism (critical for fitness content)?
4. Which tools handle close-up facial expressions without uncanny valley artifacts?
5. What's the current state of AI-generated fitness/workout content specifically?

## Category 2: Character Persistence & Multi-Shot Consistency

6. Which tools support persistent characters across multiple shots/scenes (like Higgsfield's Soul Cast)?
7. How do character consistency features compare: Soul Cast vs Runway's Act-One vs Kling's face-lock vs others?
8. Can any tool maintain wardrobe, lighting, and environment consistency across a 60-second reel (multiple cuts)?
9. What's the best approach for maintaining a consistent "brand character" across weeks of content?

## Category 3: Cinematic Capabilities

10. Which tools support cinematic aspect ratios (2.39:1, 16:9)?
11. What camera movement controls exist (dolly, crane, tracking, handheld)?
12. Which tools support start-frame + end-frame video generation (keyframe interpolation)?
13. What's the maximum clip duration per tool? Can any produce >15 seconds continuously?
14. Which tools support color grading, either built-in or via API?

## Category 4: API & Programmatic Access

15. Which tools have public REST APIs for programmatic video generation?
16. Which have official SDKs (Python/Node)?
17. Which have MCP servers (community or official)?
18. What are the rate limits and queue times for API-based generation?
19. Can any be fully orchestrated from Claude Code without touching a browser?

## Category 5: Cost & Economics

20. What is the cost per minute of generated video for each tool?
21. What are the subscription tiers and credit systems?
22. Which tools offer the best quality-to-cost ratio for regular content production (4-7 posts/week)?
23. Are there free tiers or trial credits available for testing?

## Category 6: Instagram & Social Media Optimization

24. Which tools natively support Instagram-optimized export (1080x1920 9:16, H.264)?
25. Do any tools have built-in social media publishing or scheduling?
26. Which tools are being used by successful fitness/lifestyle Instagram creators right now?
27. What posting cadence is realistic if we're generating all content via AI?

## Category 7: Workflow & Integration

28. Can any tool handle the full pipeline: script → frames → video → grade → export?
29. Which tools integrate with FFmpeg, After Effects, DaVinci Resolve, or CapCut for post-production?
30. What's the fastest end-to-end workflow from prompt to Instagram-ready video?

## Category 8: Emerging & Dark Horse Tools

31. Are there any new tools (launched Q1 2026) that the market hasn't fully discovered yet?
32. Are there open-source alternatives (Wan2.1, CogVideo, etc.) worth self-hosting for cost savings?
33. What's the state of real-time AI video generation (for live content)?

## Priority Ranking
- **MUST ANSWER**: Questions 1-2, 6-8, 15-17, 20-22, 26
- **SHOULD ANSWER**: Questions 3-5, 9-14, 18-19, 24-25, 28-30
- **NICE TO HAVE**: Questions 23, 27, 31-33
