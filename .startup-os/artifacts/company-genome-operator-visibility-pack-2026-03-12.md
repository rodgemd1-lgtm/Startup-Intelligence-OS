# Company Genome Operator Visibility Pack

**Date:** 2026-03-12  
**Owner:** Jake

## Objective
Make company, project, decision, and run state visible from the operator layer instead of leaving company genome objects buried in files.

## Visible surfaces
- `.startup-os/workspace.yaml`
- `.startup-os/companies`
- `.startup-os/projects`
- `.startup-os/decisions`
- `.startup-os/artifacts/department-maturity-dashboard-2026-03-12.md`
- `apps/decision_os`
- `apps/v5`

## Required operator widgets
- active company and project context
- active decision context
- linked department maturity
- latest run summaries
- newest artifacts by company and project

## Success condition
An operator can see the state of the company genome without opening raw YAML first.
