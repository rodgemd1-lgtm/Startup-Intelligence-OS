# Persona and Agent Evaluation Pack

**Date:** 2026-03-12

## Objective
Give personas and routed agents a shared evaluation layer so access is judged by output quality, not only by available memory.

## Evaluation questions
- Did the persona use the right corpus?
- Did the agent cite named evidence?
- Was the handoff to the next department coherent?
- Did the output improve a downstream decision or artifact?
