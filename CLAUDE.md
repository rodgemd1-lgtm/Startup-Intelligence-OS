# Startup Intelligence OS

## Project Overview
Curated resource hub for AI startup founders. 23 categories covering product, engineering, growth, AI frameworks, decentralized AI, Claude Code mastery, and more. Primarily markdown — no application code.

## Structure
- Each category has its own directory with a `README.md`
- `LIVE-RESEARCH-ENRICHMENT.md` — Latest live research across all categories
- `claude-plugins/` — Vendored Claude Code plugin collection (do not edit)
- `claude-code-mastery/` — Claude Code best practices and configuration guides

## Categories
agent-skills-and-nesting, ai-approaches-frameworks, ai-product-development, algorithm-rd, audience-intel, automation, brand-positioning, claude-code-mastery, customer-success, data-analytics, decentralized-ai, engineering, finance, foundation-legal, growth-scaling, marketing, operations, product-building-management, revenue, sales-conversion, security-privacy, team-hiring, ux-ui-design, zero-to-10m-playbook

## Content Standards
- Resources include: name, description, URL, and discovery date where applicable
- Use markdown tables for resource listings
- Each README follows a consistent section structure within its category
- URLs must be verified as live before adding
- Cross-reference `LIVE-RESEARCH-ENRICHMENT.md` to avoid duplicates
- When adding new resources, check existing entries first

## Research Workflow
- **Exa AI** — Semantic search for discovering resources
- **Firecrawl** — Deep web crawling of key articles and repositories
- **Apify** — Web scraping for structured data extraction
- Always deduplicate against existing content across all READMEs
- Tag research with date for freshness tracking

## Commands
- No build/test/lint commands — this is a knowledge base
- Use `git` for version control
