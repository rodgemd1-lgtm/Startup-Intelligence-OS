# Customer Success — Retention, Onboarding & Expansion Playbook

> Reduce churn, increase NPS, and drive expansion revenue from 0 → $10M.

---

## 1. Onboarding Framework

### The First 7 Days (Critical Window)
| Day | Action | Goal |
|-----|--------|------|
| 0 | Welcome email + quickstart guide | Set expectations |
| 1 | Interactive setup wizard | Core feature activated |
| 2 | "Did you know?" feature email | Expand usage |
| 3 | Check-in (automated or personal) | Remove blockers |
| 5 | Success story / use case email | Inspire deeper use |
| 7 | NPS survey + usage review | Measure activation |

### Activation Metrics
- Define YOUR "aha moment" (e.g., "user completes first AI analysis")
- Track time-to-activation (target: < 24 hours)
- Users who activate in first 48 hours → 3x more likely to convert

### Onboarding Tools
| Tool | Purpose |
|------|---------|
| **Intercom** | In-app messaging, tours, chatbot |
| **Appcues** | Product tours, checklists, NPS |
| **UserGuiding** | No-code onboarding flows |
| **Loom** | Video walkthroughs |
| **Canny** | Feature request tracking |

## 2. Customer Health Scoring

### Health Score Components
| Signal | Weight | Measurement |
|--------|--------|-------------|
| **Product usage** | 30% | DAU/WAU ratio, feature depth |
| **Feature adoption** | 20% | % of features used |
| **Support tickets** | 15% | Volume, severity, sentiment |
| **NPS/CSAT** | 15% | Latest survey score |
| **Engagement** | 10% | Email opens, community participation |
| **Payment health** | 10% | On-time payments, failed charges |

### Health Tiers
| Tier | Score | Action |
|------|-------|--------|
| Healthy | 80-100 | Upsell opportunity, ask for referrals |
| Neutral | 50-79 | Proactive check-in, education |
| At Risk | 20-49 | Immediate outreach, escalate |
| Critical | 0-19 | Executive intervention, save plan |

## 3. Churn Reduction Playbook

### Leading Indicators of Churn
1. Login frequency dropping (7-day trend)
2. Key feature usage declining
3. Support ticket sentiment turning negative
4. No logins in 14+ days
5. Failed payment not resolved in 7 days
6. Key champion left the company

### Churn Prevention Actions
| Trigger | Automated Action | Human Action |
|---------|-----------------|-------------|
| Usage drop 30%+ | In-app message + email | CS outreach call |
| No login 7 days | Re-engagement email series | Personal email from CS |
| No login 14 days | "We miss you" + incentive | Phone call |
| Failed payment | Dunning email sequence (3 emails) | Personal outreach after 2nd failure |
| NPS detractor | Immediate feedback request | CS manager call within 24hrs |
| Champion left | Alert CS team | Reach out to new stakeholder |

### Churn Benchmarks
| Metric | Good | Great |
|--------|------|-------|
| Monthly logo churn | < 5% | < 2% |
| Monthly revenue churn | < 3% | < 1% |
| Net revenue retention | > 100% | > 120% |
| Annual logo churn | < 30% | < 15% |

## 4. NPS & Feedback System

### NPS Implementation
- Send NPS survey every 90 days
- Trigger after key milestones (first month, feature adoption)
- Follow up on every response:
  - **Promoters (9-10)**: Ask for review, referral, case study
  - **Passives (7-8)**: Ask "What would make us a 10?"
  - **Detractors (0-6)**: Immediate human follow-up

### CSAT (Customer Satisfaction)
- Trigger after support interactions
- Target: > 90% satisfaction
- Track trends by agent, category, feature

## 5. Expansion Revenue

### Expansion Playbook
| Trigger | Expansion Motion |
|---------|-----------------|
| Approaching usage limit | Upgrade tier prompt |
| Adding team members | Seat-based upsell |
| Using advanced features | Premium tier conversation |
| Successful ROI demonstrated | Annual contract + expansion |
| New use case identified | Cross-sell opportunity |
| Company growing (funding, hiring) | Enterprise conversation |

### Net Revenue Retention (NRR) Targets
| Stage | Target NRR |
|-------|-----------|
| Pre-$1M ARR | > 100% (just avoid net churn) |
| $1M-$5M ARR | 110-120% |
| $5M-$10M ARR | 120-140% |
