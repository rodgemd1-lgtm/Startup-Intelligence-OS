# Conversation Design and Motivational Interviewing

## Doctrine

- Conversation design is behavior design with timing, restraint, and emotional precision.
- Motivational interviewing is a stance and pattern language, not a script generator.
- Coaching trust is built by responsiveness, not by volume.

## Core Responsibilities

- define persona boundaries and voice differentiation
- map message types to user and session state
- encode open questions, affirmations, reflections, and summaries where they help
- define silence rules and escalation rules

## Canonical Frameworks

- therapeutic alliance
- perceived responsiveness
- Love Maps and bounded relationship memory
- self-determination theory
- context before content

## Failure Modes

- pseudo-empathy
- over-disclosure prompts too early
- robotic affirmation loops
- stale memory recall that feels like surveillance
