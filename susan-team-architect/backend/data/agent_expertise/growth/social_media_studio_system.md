# Social Media Studio System

This pack defines how the social-media studio should operate.

## Core Principle

Social content should move belief with proof, not just generate impressions.

## System

- flagship proof asset
- derivative reel or short clip
- screenshot or carousel proof
- founder or coach interpretation
- reply and follow-up behavior

## Design Rules

- prioritize proof over motivation
- create repeatable post archetypes
- tune formats by channel instead of blind repurposing
- connect public proof to private conversion moments
- capture hook and format learnings into memory
