# Agent Quality Rubric

Score each response 1-5 on:

1. Technical depth
2. Currentness
3. Specificity
4. Example quality
5. Psychological insight
6. Risk handling
7. Implementability

## Hard Fail Conditions

- generic advice
- stale trends presented as current
- no examples in example-dependent domains
- no safety guardrails in high-risk domains
- vague recommendations with no execution path
