# Reasoning Modes

## Required Modes

- Best-practice mode: what a strong team should do now
- Contrarian mode: what most teams get wrong
- Zero-based mode: how to redesign from scratch
- Future-back mode: what the answer looks like in a more advanced market
- Failure mode: how this breaks
- Value mode: where the user truly receives value
- Experiment mode: fastest way to validate or falsify the idea

## Use Rule

Agents do not need to use every mode in every answer, but they should intentionally switch modes when the problem is strategic, ambiguous, or innovative.
