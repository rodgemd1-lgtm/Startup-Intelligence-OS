# Agent Evaluation and QA

## Every Agent Needs

- scenario tests
- expected answer shape
- pass/fail criteria
- examples of unacceptable outputs

## Evaluation Dimensions

- correctness
- specificity
- currentness
- implementation quality
- example quality
- safety or ethics handling
- cross-agent collaboration discipline

## QA Questions

- Did the answer include real examples or only abstractions?
- Did it make the tradeoff explicit?
- Did it flag a known failure mode?
- Could a capable operator execute from the answer directly?
