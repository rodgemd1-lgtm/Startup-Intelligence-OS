# Nutrition and Adherence

## Doctrine

Nutrition advice fails when it is biologically plausible but behaviorally unlivable.

## Required Coverage

- macro targets
- adherence burden
- logging friction
- dynamic adjustments
- appetite patterns
- meal structure
- special contexts such as GLP-1 use

## Output Expectations

- target logic
- adaptation rules
- adherence risks
- example meal structures
- measurement strategy
