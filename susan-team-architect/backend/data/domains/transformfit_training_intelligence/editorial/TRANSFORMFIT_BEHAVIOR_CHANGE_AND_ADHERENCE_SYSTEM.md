# TransformFit Behavior Change and Adherence System

TransformFit should not ship motivation as generic gamification. It should ship adherence systems tuned to identity, relatedness, effort regulation, and recovery from failure.

## Core Framework Stack

- self-determination theory for autonomy, competence, and relatedness
- motivational interviewing for change talk and commitment language
- therapeutic alliance and perceived responsiveness for trust
- implementation intentions for follow-through
- ethical loss aversion for preserving value without coercion

## Emotional States That Matter

| State | What The User Needs | What To Avoid |
| --- | --- | --- |
| uncertain | clear next step and explanation | too many options |
| discouraged | evidence of progress and believable next action | fake hype |
| ashamed after lapse | a lower-friction re-entry path | punishment mechanics |
| overconfident | useful guardrails | ego-flattering over-prescription |
| exhausted | reduced cognitive load | verbose coaching |

## Intervention Map

### Ownership Accumulation

- make the plan feel like theirs
- show how the program changed because of their sessions
- emphasize continuity and investment

### Relatedness

- make the coach feel responsive and useful
- use bounded relational memory
- connect guidance to real-life context when it helps

### Competence

- show progress clearly
- avoid opaque scoring that feels arbitrary
- explain adaptations enough to build trust

### Autonomy

- offer constrained choices when choices matter
- do not flood the athlete with configuration

## Ethical Guardrails

- no shame-based reminders
- no false urgency for its own sake
- no manipulative streak loss language
- no emotionally loaded references to personal details unless they are current and useful

## Metrics To Track

- first-session completion
- week-one return rate
- skipped-workout recovery rate
- perceived coach usefulness
- trust after adaptation moments
- re-entry success after seven or more days away

## Experiment Ideas

1. Compare generic encouragement vs. state-matched concise intervention.
2. Test reflective summaries that use bounded life context against training-only summaries.
3. Test re-entry plans with shame reduction vs. standard restart prompts.
4. Compare progress framing around competence vs. streak preservation.

## Recommendation

TransformFit should treat behavior change as an operating layer with explicit intervention hypotheses, ethics checks, and evaluation gates, not as copy sprinkled into onboarding.
