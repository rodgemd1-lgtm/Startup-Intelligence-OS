# TransformFit Program Library 2026

This is the first production-grade TransformFit program catalog. It turns the studio and research work into actual plans Susan can route, compare, and adapt.

## Design Rules

- Every program needs a clear athlete fit, not just a split name.
- Programs must include progression, deload, swap, and re-entry rules.
- The first week must feel winnable and coherent.
- Main lifts should stay stable long enough for progress to feel real.
- A missed week should degrade intelligently, not collapse the block.

## Current Library

| Program | Best For | Frequency | Equipment |
| --- | --- | --- | --- |
| Full Body Foundations | beginners and true restarts | 3 days | full gym |
| Upper Lower Power Hypertrophy | intermediates chasing size + strength | 4 days | full gym |
| Scientific Push Pull Legs | high-adherence hypertrophy lifters | 6 days | full gym |
| Efficient Hypertrophy 3-Day | busy intermediate lifters | 3 days | full gym |
| Dumbbell Momentum 3-Day | home or limited-equipment users | 3 days | dumbbells + bench |

## Selection Logic

- Use Full Body Foundations for lifters with under 12 months of consistent training or anyone rebuilding after a long layoff.
- Use Upper Lower Power Hypertrophy when the athlete can train four days and wants the best mixed strength/size tradeoff.
- Use Scientific Push Pull Legs only when schedule reliability and recovery are both high.
- Use Efficient Hypertrophy 3-Day when the athlete wants hypertrophy but real life will punish high-frequency plans.
- Use Dumbbell Momentum 3-Day when equipment access is constrained but progression still matters.

## Required Program Outputs

Every TransformFit program should surface:

- athlete fit
- weekly schedule
- exercise intent
- progression rules
- deload rules
- swap logic
- missed-session and re-entry logic
- coach message cues for week one and week after disruption

## Operating Recommendation

The library should be treated as live infrastructure. New programs should only enter the catalog when they have:

- a target athlete definition
- evidence notes
- swap and re-entry rules
- coach expression guidance
- at least one evaluation hypothesis
