# Founder Trust and Governance Baseline

Every company launched from the foundry should inherit a default trust layer.

## Baseline areas

- privacy and consent
- accessibility
- security
- AI governance
- claims governance

## Launch-gate questions

- What data is collected, and why?
- What user consent is required?
- What claims require primary-source support?
- What accessibility checks block launch?
- What security baselines are mandatory before public use?

## Foundry policy

- high-stakes claims require primary-source backing
- trust-sensitive launches require explicit review by Shield and Sentinel
- accessibility is part of the release definition of done
- AI systems should have scope limits, confidence boundaries, and escalation paths
