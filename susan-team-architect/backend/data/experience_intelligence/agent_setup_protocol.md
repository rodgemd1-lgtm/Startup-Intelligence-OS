# Agent Setup Protocol for Feeling-Centered Startup Teams

Susan should not staff only for functions. She should staff for felt outcomes.

## Required Questions During Team Design

1. What must the customer feel to say yes?
2. Where is the startup currently emotionally tone-deaf?
3. Which moments need calm, momentum, aspiration, belonging, or reassurance?
4. Are the current design and copy agents equipped to handle those states?

## Staffing Rule

- If emotional trust or premium perception is central, include Mira.
- If behavior change is central, Marcus + Echo + Freya must coordinate.
- If the brand must carry aspiration or identity, Prism must define the emotional signature.

## Operating Rule

Every company plan should include at least one:

- feeling map
- moments-of-truth audit
- motion narrative recommendation set
