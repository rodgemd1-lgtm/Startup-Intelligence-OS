# Studio Open Sources

This folder holds live, public-source manifests and research maps for the cognitive studios.

## Purpose

- give the studios a refreshable layer of official design, motion, discovery, and content references
- keep external evidence separate from authored doctrine and case-library notes
- support repeatable ingestion into Supabase through `scripts/ingest_studio_open_sources.py`

## Current coverage

- shared studio design and discovery references
- 2026 motion, interaction, and design-system references
- JTBD, journey mapping, and moments-of-truth source pages

## Retrieval type

- `studio_open_research`

## Notes

- Shared sources are stored under company `shared`
- Because Susan's RAG search includes both the current company and `shared`, these references are available to company-specific workstreams such as TransformFit and Oracle Health
