# Studio Eval Rubric
## How To Score Design And Marketing Studio Outputs

Score each dimension from 1 to 5.

## Dimensions

### 1. Diagnosis quality
- Did the work correctly identify the job, friction, audience, and belief shift?

### 2. Value detection
- Did the work identify real user, business, emotional, and operational value?

### 3. Proof quality
- Are the claims visibly grounded in proof, examples, screenshots, or evidence?

### 4. Emotional precision
- Does the work match the likely user or buyer state?

### 5. Originality
- Does the work avoid generic AI or template sameness?

### 6. Concept quality
- Did the work propose a strong route, not just a polished surface?

### 7. Reuse value
- Did the work produce a reusable pattern, rule, or template?

### 8. Experiment quality
- Did the work define a realistic way to validate the recommendation?

### 9. Implementation realism
- Is the recommendation plausible given performance, accessibility, compliance, and team constraints?

## Fail conditions

Any output should fail review if it:
- has no diagnosed job
- makes major claims with no visible proof
- is emotionally tone-deaf to the audience
- is mostly generic style language
- has no validation path
- creates no reusable lesson

## Review workflow

1. Score independently
2. Identify the weakest two dimensions
3. Revise the work to improve those first
4. Save one reusable rule into studio memory

## Minimum ship threshold

- no dimension below 3
- diagnosis, proof, and implementation realism should usually be 4+
