## Design Prompt for Stitch / Galileo AI

```
Design a modern SharePoint Online Communication Site for an enterprise healthcare

Claude, ChatGPT) with a modular prompt system called "LEGO pieces."

Brand: Oracle Health

- Accent: White (#FFFFFF)

- Style: Clean, spacious, enterprise-grade — think McKinsey meets modern SaaS documentation
```
